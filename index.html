What is Linux?

Windows - Mac OS - Chrome OS

  • Proprietary
  • Developed and controlled by one company
  • Sinlge version per hardware version

Linux

  • Open Source
  • Developed by over 200 company, many non-profit organization, thousands of developers.
  • Any can modify it
  • Someone has an idea...
  • Someone has a idea for improvement, do it
  • want to optimize it for certerian usage... do it

  • to run in low power mode
